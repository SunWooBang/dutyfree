# Dutyfree - 근무표 작성 도우미

It is a service developed to help head nurses, write an efficient work schedule.
간호사의 효율적인 근무표 작성을 도와주기 위해 개발된 서비스입니다.

## 기술 스택

- React + Vite
- TailWind
- Daisyui
  
## 주요 기능

- 근무자 정보 관리
- 근무표 자동 생성
- 엑셀 파일 가져오기/내보내기
- 근무 규칙 설정

## 개발 환경 설정

```bash
npm install
npm run dev
```
